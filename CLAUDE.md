# Game Projects

모바일 게임 개발 프로젝트. 웹(Phaser 3)으로 먼저 제작 및 테스트 후, 완성도가 충분한 게임만 모바일 앱으로 출시.

## 개발 원칙

- 웹(Phaser 3)으로 먼저 프로토타입 제작
- 브라우저에서 충분히 테스트 후 모바일 빌드 결정
- 미니멀하고 중독성 있는 게임 위주
- 광고 수익 모델 (AdMob) 기반

## 기술 스택

- **프레임워크**: Phaser 3
- **언어**: JavaScript
- **모바일 변환**: Capacitor (출시 결정 시)
- **광고**: AdMob (출시 결정 시)

## 프로젝트 목록

### 2. Star Assault (`./star-assault/`)
> 상태: 설계 완료, 개발 예정

Tap Titans 스타일 클리커 + 방치형 게임. SF/우주 배경, AI 반란 세계관.
**기본 구조는 Tap Titans과 동일**하게 구현하되 테마와 콘텐츠만 Star Assault에 맞게 변경.

**세계관**
- 2387년, 인류가 만든 AI 함대 **ARES**가 반란
- 플레이어는 저항군 **Star Assault Corps** 사령관
- 최종 목표: ARES 본부 파괴

**핵심 게임플레이 (Tap Titans 동일)**
- 시점: 횡스크롤
- 탭: 화면 아무데나 탭 → 공격
- 적: 오른쪽에서 한 마리씩 다가옴
- 내 함선: 왼쪽 고정, 자동 공격
- 보스: 매 스테이지 마지막 미니보스 + 5스테이지마다 진짜 보스 (제한시간 30초)
- 보스 실패 시: 이전 스테이지로 후퇴 (광고 보고 부활 가능)

**화폐 시스템**
| 화폐 | 획득 | 용도 | 지속성 |
|------|------|------|--------|
| 골드 | 적 처치, 오프라인 보상 | 영웅 고용/레벨업, 펫 알 구매 | 런 내 |
| 스타더스트 | Prestige 시 획득 | 스킬트리, 유물 레벨업 | 영구 유지 |

**영웅 시스템 (Tap Titans 동일)**
- 15명+, 목록에서 고용/레벨업
- 레벨 25/50/100/200 달성 시 패시브 스킬 해금
- 액티브 스킬 6개 (쿨타임 있음):
  1. Nova Blast - 탭 데미지 10배 (30초) / 쿨 3분
  2. Warp Strike - 보스 HP 5% 즉시 제거 / 쿨 5분
  3. Overdrive - 자동 공격 속도 3배 (20초) / 쿨 4분
  4. Gold Rush - 골드 획득 10배 (30초) / 쿨 6분
  5. Shield Surge - 보스 제한시간 +15초 / 쿨 8분
  6. ARIA Mode - 모든 스킬 쿨타임 즉시 초기화 / 쿨 10분

**영웅 목록**
| # | 이름 | 역할 |
|---|------|------|
| 1 | Vega | 탭 데미지 강화 |
| 2 | Iron Rex | 모든 데미지 +% |
| 3 | Nova | 보스 데미지 특화 |
| 4 | Kira | 골드 획득 +% |
| 5 | Titan-7 | 광역 딜 |

| 8 | Orion | 모든 영웅 강화 |
| 9 | Blaze | 탭 크리티컬 데미지 |
| 10 | Specter | 스킬 쿨타임 감소 |
| 11 | Atlas | 자동 딜 특화 |
| 12 | Lyra | 스타더스트 획득 +% |
| 13 | Phantom | 골드 + 탭 혼합 |

**Prestige 시스템 (Tap Titans 동일)**
- 100스테이지 도달 시 활성화
- 리셋 + 스타더스트 획득
- 스킬트리 (Offense / Support / Economy)
- 유물 시스템: Prestige마다 랜덤 획득, 스타더스트로 레벨업 (15종)

**유물 목록**
ARES 코어, 양자 배터리, 나노 장갑, 스텔스 칩, 과부하 모듈, 에너지 증폭기,
전투 알고리즘, 정찰 드론, 반중력 엔진, ARIA 회로, 광자 캐논, 시간 왜곡기,
암흑물질 코어, 행성 방어막, 은하 지도

**펫 시스템 (Tap Titans 동일)**
- 골드로 알 구매 → 부화 → 펫 획득
- 중복 시 강화, 한 번에 1마리 장착
- 펫마다 고유 액티브 스킬

**스테이지 구역 (Tap Titans 동일 구조)**
- 구역별 배경 변화 (소행성대 → 붉은 성운 → 암흑 구역 → ARES 점령지 → ARES 본부)

**오프라인 보상**
- 최대 12시간치 골드, 광고 보면 2배

**비주얼 (Tap Titans 스타일)**
- 타격 시 데미지 숫자 팡팡
- 크리티컬 시 더 크고 화려한 숫자
- 적 폭발 파편 이펙트
- 레이저/미사일 발사 이펙트
- 하단 영웅 목록 스크롤 UI

**수익화**
- 광고 (AdMob): 오프라인 보상 2배, 보스 부활, 스타더스트 보너스, 영웅 임시 버프
- 인앱결제: 스타터 팩, 월정액 패스 (매일 스타더스트 + 광고 제거)

**튜토리얼**
- 강제 없이 플레이하면서 자연스럽게 배우는 구조 (Tap Titans 동일)

**추후 업데이트 예정**
- 토너먼트 (주간 랭킹)
- 클랜 시스템

### 1. Dot Dodge (`./dot-dodge/`)
> 상태: 개발 중

SF 우주 배경의 네모 피하기 스테이지 생존 게임.

**핵심 게임플레이**
- 플레이어: 시안색 네모, HP 숫자 표시
- 적: 데미지 숫자 표시된 네모, 스테이지마다 강해짐
- 조작: 가상 조이스틱 (드래그 시작 위치에 생성)
- 스테이지당 15초 버티면 클리어
- 10스테이지마다 보스 등장
- 비주얼: 우주 배경 + 별 + 시안 글로우

**화폐 시스템**
| 화폐 | 획득 | 용도 | 지속성 |
|------|------|------|--------|
| 캐시 | 적 처치 | 스테이지 클리어 업그레이드 | 런 초기화 |
| 코인 | 보스 처치 / 특별 이벤트 | 영구 업그레이드 + 카드 뽑기 | 영구 유지 |

**카드 시스템 (효력카드)**
- 코인으로 뽑기
- 런 시작 전 장착하는 패시브 버프
- 중복 획득 시 강화 (1성 → 2성 → 3성)
- 카드 예시: 흡혈, 코인 획득량 증가, 방어막, 캐시 획득량 증가 등

**스테이지 클리어 보상**
- HP 20% 회복
- 업그레이드 3개 중 선택 (캐시 소모): Max HP, 방어력, HP 회복

**광고 포인트**
- 게임오버 시 HP 회복 (부활)
- 업그레이드 시간 단축 (추후)
