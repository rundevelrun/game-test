# Game Projects

모바일 게임 개발 프로젝트. 웹(Phaser 3)으로 먼저 제작 및 테스트 후, 완성도가 충분한 게임만 모바일 앱으로 출시.

---

## 개발 원칙

- 웹(Phaser 3)으로 먼저 프로토타입 제작
- 브라우저에서 충분히 테스트 후 모바일 빌드 결정
- 미니멀하고 중독성 있는 게임 위주
- 광고 수익 모델 (AdMob) 기반

## 기술 스택

- **프레임워크**: Phaser 3.60 (CDN)
- **언어**: JavaScript (ES6)
- **배포**: GitHub Pages (`rundevelrun/game-test` 레포)
- **도메인**: `game.rundevelrun.com` (Cloudflare DNS → GitHub Pages)
- **모바일 변환**: Capacitor (출시 결정 시)
- **광고**: AdMob (출시 결정 시)

## 게임 1: Dot Dodge (`./dot-dodge/`)

> **상태**: 개발 중 (기본 동작 완료)

SF 우주 배경의 네모 피하기 스테이지 생존 게임.

### 핵심 게임플레이
- 플레이어: 시안색 네모, HP 숫자 표시
- 적: 데미지 숫자 표시된 네모, 스테이지마다 강해짐
- 조작: 가상 조이스틱 (드래그 시작 위치에 생성)
- 스테이지당 15초 버티면 클리어
- 10스테이지마다 보스 등장
- 비주얼: 우주 배경 + 별 + 시안 글로우

### 화폐 시스템
| 화폐 | 획득 | 용도 | 지속성 |
|------|------|------|--------|
| 캐시 | 적 처치 | 스테이지 클리어 업그레이드 | 런 초기화 |
| 코인 | 보스 처치 / 특별 이벤트 | 영구 업그레이드 + 카드 뽑기 | 영구 유지 |

### 카드 시스템 (미구현)
- 코인으로 뽑기
- 런 시작 전 장착하는 패시브 버프
- 중복 획득 시 강화 (1성 → 2성 → 3성)

### 스테이지 클리어 보상
- HP 20% 회복
- 업그레이드 3개 중 선택 (캐시 소모): Max HP, 방어력, HP 회복

### 미구현 항목
- 코인 시스템 (획득/사용)
- 카드 뽑기 시스템
- 보스 구현

## 게임 2: Star Assault (`./star-assault/`)

> **상태**: 개발 중 (핵심 루프 완료)

Tap Titans 스타일 클리커 + 방치형 게임. SF/우주 배경, AI 반란 세계관.
**기본 구조는 Tap Titans과 동일**하게 구현하되 테마와 콘텐츠만 Star Assault에 맞게 변경.

### 세계관
- 2387년, 인류가 만든 AI 함대 **ARES**가 반란
- 플레이어는 저항군 **Star Assault Corps** 사령관
- 최종 목표: ARES 본부 파괴
- ARIA: 적 AI에서 아군으로 전향한 특별 영웅

### 현재 구현된 것
- 횡스크롤 전투 (내 함선 왼쪽 고정, 적 오른쪽에서 진입)
- 화면 탭 → 공격 (데미지 팝업, 크리티컬)
- 영웅 8명 고용/레벨업 (하단 스크롤 목록)
- **마일스톤 보너스**: 레벨 10/25/50/100/200 도달 시 DPS 폭발적 증가
- **일괄 구매**: x1 / x10 / Max 버튼
- **킬 카운터**: 전투 영역에 X/10 표시
- **다음 마일스톤 표시**: `Lv.8 → 10★` 형식
- 골드 시스템 (K/M/B/T 포맷)
- 미니보스 (매 스테이지 끝, HP x10, 30초 타이머)
- 진짜 보스 (5스테이지마다, HP x50, 30초 타이머)
- 보스 실패 시 이전 스테이지 후퇴
- 액티브 스킬 4개 (Nova Blast, Warp Strike, Overdrive, Gold Rush)
- localStorage 자동저장 (5초마다)

### 핵심 게임플레이 설계 (Tap Titans 동일)
- 탭: 화면 아무데나 탭 → 공격
- 적: 오른쪽에서 한 마리씩 다가옴, 10마리 처치 → 다음 스테이지
- 내 함선: 왼쪽 고정, 자동 공격 (영웅 DPS)
- 보스: 매 스테이지 마지막 미니보스 + 5스테이지마다 진짜 보스 (30초)
- 보스 실패 시: 이전 스테이지 후퇴 (광고 보고 부활 가능)

### 스케일링 공식
```
적 HP = 10 * 1.4^(stage-1)
미니보스 HP = 적 HP * 10
보스 HP = 적 HP * 50
골드 = stage * 8 * 1.12^(stage-1)
영웅 비용(lv) = baseCost * 1.07^lv
영웅 DPS(lv) = baseDps * 1.15^(lv-1) * lv * milestoneMultiplier
탭 데미지 = totalDPS * 0.5 (최소 1), 10% 크리티컬 (2배)
```

### Prestige 시스템 (미구현)
- 100스테이지 도달 시 활성화
- 리셋 + 스타더스트 획득
- 스킬트리 (Offense / Support / Economy)
- 유물 시스템: Prestige마다 랜덤 획득, 스타더스트로 레벨업

### 유물 목록 (15종, 미구현)

전투 알고리즘, 정찰 드론, 반중력 엔진, ARIA 회로, 광자 캐논, 시간 왜곡기,
암흑물질 코어, 행성 방어막, 은하 지도
